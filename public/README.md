# Pasta Public

Esta pasta contém arquivos estáticos que serão servidos diretamente.

## Logo da MedMais

Para adicionar o logo na tela de login, coloque o arquivo do logo aqui com o nome:

**`logo-medmais.png`** (ou `.svg`, `.jpg`)

O arquivo será automaticamente carregado em `/logo-medmais.png` na aplicação.

### Formatos suportados:
- PNG (recomendado para logos com transparência)
- SVG (melhor qualidade e escalabilidade)
- JPG (se não tiver transparência)

### Tamanho recomendado:
- Altura: 80-120px
- Largura: proporcional ao logo
- Resolução: 2x para telas retina (ex: 240px de altura se usar 120px no CSS)
